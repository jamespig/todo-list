# Todo List 專案

簡潔的待辦事項管理應用程式。

## 目錄
- [技術棧](#技術棧)
- [功能](#功能)
- [開發](#開發)
- [部署](#部署)
- [貢獻](#貢獻)
- [授權](#授權)

## 技術棧

- **前端框架**: React 18
- **語言**: TypeScript
- **構建工具**: Vite
- **樣式**: Tailwind CSS, SASS
- **拖放功能**: react-beautiful-dnd
- **圖標**: lucide-react

## 功能

- 待辦事項的創建、讀取、更新和刪除
- 任務狀態管理（完成/未完成）
- 拖放排序功能
- 響應式設計

## 開發

```bash
# 安裝依賴
npm install

# 啟動開發服務器
npm run dev

# 構建生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 部署

此專案使用 GitHub Pages 進行部署：

```bash
npm run deploy
```
